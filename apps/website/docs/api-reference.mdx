# Provider API

[Z3US](https://z3us.com) wallet extension injects a global API into websites visited by its users at `window.z3us`. This API allows users to interact with wallet and [Radix DLT](https://www.radixdlt.com/).

## Versioning

API is versioned using [Semantic Versioning](https://semver.org/). We will support up to 2 most recent versions to ensure users have time to update when breaking changes are introduced.
