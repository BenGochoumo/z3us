# Introduction

Welcome to the [Z3US](https://z3us.com) documentation. Here you will find information on how to use the wallet or interact with [Z3US](https://z3us.com) public API.

## Why Z3US ?

[Z3US](https://z3us.com) was created to impvrove user experiance while interacting with [Radix DLT](https://www.radixdlt.com/). It purpose is to ensure your security while being fully transparent.

Our codebase is opensource on [GitHub](https://github.com/z3us-dapps).
